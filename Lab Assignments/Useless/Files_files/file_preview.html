Activity 1: (2 pts.) (Software Installation) Do the following tasks:
i. Download R from r-project.org
ii. Download RStudio from rstudio.com
iii. Install these packages: tidyverse, arules, knitr, caret, markdown, reticulate.
If you are using RStudio Server, you might encounter some error messages when trying
to install a new package. If, for example, you are trying to install the tidyverse package and
you get an error message, then try the following command:
install.packages("tidyverse", dependencies = TRUE, IN-
STALL_opts = '--no-lock')
After installing these packages, write in your markdown the output of the following R
commands (type them in markdown, not in the terminal)
inst.packages <- installed.packages()[,1]
all(is.element(c("tidyverse", "arules" , "caret", "markdown", "reticulate",),
inst.packages))
The output must be True.

a) Download the adult dataset (https://archive.ics.uci.edu/ml/datasets/adult) from
UCI.

c) (5 pts.) Read the file adult.data containing the training data into the variable ‘adult’.

d) (2 pts.) Write and execute the command to find H t , the transpose of the matrix H of

multiplication. The resulting matrix must be of size 2x2.
f) (2 pts.) Write and execute the command to find the inverse of the matrix H t x H.
g) (2 pts.) Write and execute the command to verify that the matrix found in part f is
indeed the inverse of H t x H.
